sg 2021-03-11
  • 变化：两者作用于同一个元素上时，v-if 会拥有比 v-for 更高的优先级。

# 2.x 语法

2.x 版本中在一个元素上同时使用 v-if 和 v-for 时，v-for 会优先作用。

# 3.x 语法

3.x 版本中 v-if 总是优先于 v-for 生效。
